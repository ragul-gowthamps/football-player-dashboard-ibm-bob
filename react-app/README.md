# Player Dashboard — React App

A Premier League player dashboard built with **React** and the **Carbon Design System**.

## Features

| Page | Description |
|---|---|
| **Player Browser** | Search, filter by position, sort, select a player to see their stats and summary |
| **Compare Players** | Pick two players side-by-side to compare their stats |
| **Team Formation** | Choose a formation (4-4-2, 4-3-3, 3-5-2), generate a position-aware squad, view it on an interactive pitch with hover tooltips and click-to-inspect |

## Project structure

```
react-app/
├── public/               Static assets
├── scripts/
│   └── fetchPlayers.mjs  Data fetch script (requires Node + API key)
├── server/
│   └── server.js         Express server (for production build)
├── src/
│   ├── components/       All React components
│   ├── data/             players.json (generated by fetch script)
│   ├── utils/            teamGenerator.js
│   ├── App.jsx           Main app with navigation
│   └── index.js          Entry point
├── .env                  API key (not committed)
├── package.json
└── yarn.lock
```

## Setup

### 1. Install dependencies
```bash
yarn install
```

### 2. Add your API key
Create a `.env` file in this folder:
```
REACT_APP_API_FOOTBALL_KEY=your_key_here
```

### 3. Fetch player data
```bash
node scripts/fetchPlayers.mjs
```
This creates `src/data/players.json`.

### 4. Start the dev server
```bash
yarn start:dev
```
Opens at **http://localhost:3000**.

## Build for production
```bash
yarn build
node server/server.js
```
